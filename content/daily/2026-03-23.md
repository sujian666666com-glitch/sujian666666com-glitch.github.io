## 今日要闻

### 1. Flash-MoE：笔记本运行 397B 参数模型
GitHub 上开源的项目 Flash-MoE 展示了如何在消费级笔记本电脑上运行 3970 亿参数的 MoE（混合专家）模型。这是一个重要的技术突破，使得大模型推理不再局限于高端服务器。

### 2. Project Nomad：离线知识库
Project Nomad 获得 300+ Hacker News 点数，目标是构建永不离线的知识管理系统。在网络不稳定或需要数据主权控制的场景下具有重要价值。

### 3. Windows 原生应用开发现状
一篇文章深入分析了 Windows 原生应用开发的现状，指出开发体验碎片化、工具链不稳定等问题，获得 263 点关注。

### 4. .NET MAUI 将支持 Linux
Avalonia 团队宣布 MAUI 将迎来 Linux 支持，这对跨平台 .NET 开发者是好消息。

### 5. Palantir 扩展英国政府数据访问
据 The Guardian 报道，Palantir 获得了英国 FCA（金融行为监管局）敏感数据的访问权限，引发隐私和数据安全讨论。

### 6. 版本控制的未来
Bram Cohen（BitTorrent 作者）撰文探讨版本控制系统的未来发展方向。

### 7. 代码不会消亡
一篇名为 "Reports of code's death are greatly exaggerated" 的文章获得 142 点，认为 AI 不会让程序员失业，而是改变编程方式。
