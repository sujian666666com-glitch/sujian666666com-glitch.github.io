## 什么是 OpenClaw Skill？

在深入之前，先理解 Skill 的本质：

**Skill = 专业领域知识 + 工具使用指南 + 最佳实践**

每个 Skill 是一个包含 `SKILL.md` 文件的目录，其中定义了：
- **Name**：技能名称
- **Description**：一句话描述
- **Metadata**：运行环境要求（依赖的二进制、环境变量、配置项）
- **Instructions**：详细的工具使用指南和最佳实践

OpenClaw 在加载时会扫描所有 Skill，并将符合条件的技能注入到 AI 的系统提示词中。这意味着：
- ✅ **零代码**：不需要写任何代码就能扩展 AI 能力
- ✅ **热插拔**：安装即用，卸载即停
- ✅ **可定制**：可以覆盖官方 Skill，打造专属能力
- ✅ **安全可控**：通过元数据控制加载条件

## 如何开始使用 Skill？

### 方法 1：通过 ClawHub 安装（推荐）

```bash
# 安装 PostgreSQL Queries Skill
npx clawhub@latest install pg-queries

# 安装 Home Assistant Control Skill
npx clawhub@latest install hass-control
```

### 方法 2：手动创建 Skill

在 `~/.openclaw/skills/` 目录下创建 `my-skill/SKILL.md`：

```yaml
---
name: my-custom-skill
description: 我的自定义技能
metadata: {
  "openclaw": {
    "requires": {
      "bins": ["python3"]
    }
  }
}
---

2. 先测试再执行
3. ...
```

### 方法 3：从 GitHub 导入

```bash
# 克隆 Skill 仓库到本地
git clone https://github.com/example/openclaw-skills.git ~/.openclaw/skills/
```
