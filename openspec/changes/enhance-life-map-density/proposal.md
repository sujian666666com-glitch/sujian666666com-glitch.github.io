# LifeMap 密度与叙事节奏改造

## 目标

将 LifeMap v2 从「密集节点表格地图」调整为「松散、可阅读、有叙事节奏的手账冒险地图」，保留暖色手账风格、双模式、青春支线和数据保护。

## 范围

- 地图节点减密度：按 type / visualKind 分形渲染，地图仅保留短标题与章节
- 布局拉开：modePosition 坐标扩大并分区（家庭左、成长右、青春右上弧）
- 详情面板改手账页：段落 + 便签 + 情绪标签 + 故事小路
- 默认分层展示：青春支线 ghost，点击「她」聚焦完整小路
- Canvas fitView / minZoom 适配更大画布

## 不做什么

- 不改密码验证与 `/api/life-map/*` 契约
- 不把真实数据移到客户端 bundle
- 不改节点 summary / 敏感字段内容

## 验收标准

- 第一眼像松散手账冒险地图，而非小表格堆叠
- 地图节点可扫读，无 period/status/长说明
- 点击「她」后青春支线成为独立剧情小路
- 详情面板像手账页，无字段表格感
- 320px 无横向溢出
